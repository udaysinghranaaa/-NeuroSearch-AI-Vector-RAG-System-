# NeuroSearch AI (Vector + RAG System)

A C++ based vector database with semantic search and Retrieval-Augmented Generation (RAG) using a local LLM (Ollama).

## 🚀 Features

* Vector database with HNSW, KD-Tree, and brute-force search
* Semantic search using embeddings
* RAG pipeline with local LLM (Ollama)
* Interactive UI for querying and visualization

## 🛠 Tech Stack

* C++
* HTML, CSS, JavaScript
* Ollama

## ▶️ Run Locally

1. Start Ollama:
   ollama serve

2. Run project:
   ./db

3. Open:
   http://localhost:8080
